给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

最长回文子串动态规划法

回文串因为其自生的原因就可以写出动态转移方程：
P(i, j) = P(i + 1, j - 1) && (s[i] == s[j])
其中P(i, j) 的值为true或者false，当S[i..j]为回文串时，其值为true，否则为false。

最长回文子串中心扩散法

每一个回文串都可以看作是由回文串的中心字符向两边的扩展，那么我们对字符串中的每一个字符都进行奇数串和偶数串的扩展，最后记下最长的那个子串，即为所求的子串。此方法空间复杂度为O(1)。
