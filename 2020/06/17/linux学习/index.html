文件处理命令

命令格式与目录处理命令ls

ls -a 列出目录所有文件，包含以.开始的隐藏文件
ls -A 列出除.及..的其它文件
ls -r 反序排列
ls -t 以文件修改时间排序
ls -S 以文件大小排序
ls -h 以易读大小显示
ls -l 除了文件名之外，还将文件的权限、所有者、文件大小等信息详细列出来
ls -i iNode

目录处理命令

文件处理命令

touch

命令所在路径：/bin/touch
执行权限：所有用户
语法:touch [文件名]
功能描述：创建空文件
范例：touch haha.list touch “program files”

cat tac

命令所在路径：/bin/cat
语法：cat [文件名]
功能描述：显示文件内容
-n 显示行号
范例: cat /etc/issue
cat -n /etc/services
tac 为反向显示

more

语法: more [文件名]
功能描述：分页显示文件内容
实例： more /etc/services
使用 space或f 翻页
enter 换行
q或Q 退出

less

功能描述：向上翻页或查找

head

功能描述：查看文件的前几行
语法：head -n 行数 文件路径
默认为10行

tail

功能描述：查看文件的前几行
语法：tail -n 行数 文件路径
tail -f 动态显示文件内容

链接命令

ln

命令英文原意：link
命令所在路径：/bin/ln
语法：ln -s [原文件] [目标文件]
-s 创建软链接（类似于windows快捷方式）
功能描述：生成链接文件
硬链接与cp -p区别：
1、硬链接文件与原文件同步更新
2、硬链接文件与原文件iNode值相同
3、硬链接不能跨分区
4、不能针对目录使用

权限管理命令

权限管理命令chmod

命令英文原意：change the permissions mode of a file
语法：chmod [{ugoa} {+-=} {rwx}] [文件或目录]
[mode=421] [文件或目录]（r = 4，w = 2，x = 1）
-R 递归修改
将目录与其下文件权限均修改
功能描述：改变文件或目录权限
注意：对文件来说 r权限说明我们可以查看文件内容 对目录来说 可以列出目录中的内容
w权限可以修改文件内容 可以在目录中创建、删除文件
x权限可以执行文件 可以进入目录

其他权限管理命令

chown

命令英文原意：change owner
语法：chown [用户][文件或目录]
功能描述：改变文件或目录的所有者（只允许root用户改变）
范例 chown cmq /root/tmp/haha

chgrp

命令功能：改变文件或目录的所属组

umask

命令英文原意：the user file-creation mask
命令所在路径：Shell内置命令
执行权限：所有用户
语法 umask [-S]
-S表示以rwx形式显示新建文件缺省权限
注意： touch的文件不具备可执行权限但mkdir的目录具有

文件搜索命令

find

命令名称：find
命令所在路径：/bin/find
执行权限：所有用户
语法：find [搜索范围][匹配条件]
功能描述：文件搜索
示例：find / -name - iname（不区分大小写） -size(参数为+多少则为大于，-多少则为小于) -user [..]
find /etc -cmin -amin -mmin(change文件属性, access访问时间, modify文件内容) -5(5分钟)
-a -o 分别为连接符表示与和或
-inum 文件的i结点
find /etc -name inittab -exec ls -l {} ;

locate

命令所在路径：/usr/bin/locate
语法：locate 文件名
功能描述：在文件资料库中查找文件
示例：locate inittab
拓展： updatedb命令用来更新文件资料库
locate -i 命令可以不区分大小写搜索文件
注：文件资料库不收录tmp文件夹下的文件

which

命令路径：/usr/bin/which
语法：which 命令
功能描述：搜索命令所在目录及别名信息
范例：which ls

whereis

功能描述：搜索命令所在目录及帮助文档路径

grep

命令所在路径：/bin/grep
执行权限:所有用户
语法：grep -iv [指定子串][文件]
功能描述：在文件中搜寻字串匹配的行并输出
-i 不区分大小写
-v 排除指定子串
范例：grep mysql /root/install.log
grep -v ^# /etc/inittab

压缩解压命令

gzip

命令英文原意：GNU zip
命令所在路径：/bin/gzip
执行权限：所有用户
语法：gzip [文件]
功能描述：压缩文件(不能压缩目录)
压缩后文件格式：.gz

gunzip

命令英文原意：GNU unzip
命令所在路径：/bin/gunzip
执行权限:所有用户
语法：gunzip [压缩文件]
功能描述：解压缩.gz的压缩文件
拓展：使用 gzip -d同理

tar

命令所在路径：/bin/tar
执行权限：所有用户
语法：tar 选项[-z -c -f -v] [压缩后文件名] [目录]
-c 打包
-v 显示详细信息
-f 指定文件名
-z 打包同时压缩
-j 打包同时压缩bzip2
功能描述：打包目录
压缩后文件格式：.tar.gz
拓展：打包和压缩的区别
相同点：打包和压缩都可以将多个文件生成一个文件存储
不同点：压缩操作不但要生成一个文件，还要通过压缩工具对文件进行压缩，来减少文件占用的存储空间大小。

tar命令解压缩

-x 解包
-v 显示详细信息
-f 指定压缩文件
-z 解压缩
-j 解压缩bzip2
例：tar -zxvf Japan.tar.gz

zip

命令所在路径：/usr/bin/zip
执行权限：所有用户
语法：zip [-r] [压缩后的文件名] [需要压缩的文件或目录]
-r 压缩目录
功能描述： 压缩文件或目录
压缩后文件格式：.zip
解压缩：unzip

bzip2

命令所在路径：/usr/bin/bzip2
执行权限：所有用户
语法：bzip [-k] [文件]
-k 产生压缩文件后保留原文件
功能：压缩文件(推荐压缩较大文件)
压缩后文件格式：.bz2
范例：tar -cjf ***.tar.bz2 ***
解压缩： bunzip2 -k [文件名.bz2]
tar -xjf [文件名.tar.bz2]

netstat

命令所在路径：/bin/netstat
执行权限：所有用户
语法：netstat [选项]

-u UDP协议

范例：netstat -tlun 查看本机监听的端口
netstat -an 查看本机所有的网络连接
netstat -rn 查看本机路由表

setup

功能描述：配置网络

mount

执行权限：所有用户
命令语法：mount [-t 文件系统] 设备文件名 挂载点
命令功能：挂载设备
范例： mount -t iso9660 /dev/sr0 /mnt/cdrom
设备卸载：umount /dev/sr0

关机重启命令

shutdown

语法：shutdown [选项] 时间选项
选项： -c 取消前一个关机命令
-h 关机(halt)
-r 重启(reboot)
例：shutdown -h now
shutdown -r now
拓展：
其他关机命令：halt poweroff init 0
其他重启命令：reboot init 6
系统运行级别：0 关机
1 单用户
2 不完全多用户，不含NFS服务(网络文件系统，用于两个linux之间的文件共享)
3 完全多用户
4 未分配
5 图形界面
6 重启
退出登录命令：logout

rpm命令管理

rpm包安装

rpm -ivh 包全名
选项： -i(install) 安装
-v(verbose) 显示详细信息
-h(hash) 显示进度
–nodeps 不检测依赖性

rpm包升级

rpm -Uvh 包全名
选项： -U(upgrade) 升级

rpm包卸载

rpm -e 包名
选项： -e(erase) 卸载
–nodeps 不检查依赖性

查询rpm包是否安装

rpm -q 包名
选项： -q 查询(query)
rpm -qa
选项： -a 所有(all)

查询软件包详细信息

rpm -qi 包名
选项： -i 查询软件信息(information)
-p 查询未安装包信息(package)

查询软件包中文件安装位置

rpm -ql 包名
选项： -l 列表(list)
-p 查询未安装包信息(package)

查询系统文件属于哪个rpm包

rpm -qf 系统文件名
选项： -f 查询系统文件属于哪个软件包(file)

查询软件包的依赖性

rpm -qR 包名
选项： -R 查询软件包的依赖性(requires)
-p 查询未安装包的信息(package)

rpm包校验

rpm -V 已安装的包名
选项： -V 校验指定RPM包中的文件(verify)
